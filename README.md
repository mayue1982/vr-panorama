# VR全景景点

一个基于A-Frame的全景VR体验网站，支持：
- EXR/HDR全景图片加载
- PNG/JPG普通全景图上传
- 触摸/鼠标拖拽查看
- 远近缩放
- 自动旋转
- VR模式

## 使用方法

直接在浏览器中打开 `index.html` 即可体验。

## GitHub Pages 部署

1. 将所有文件上传到GitHub仓库
2. 在仓库Settings → Pages中启用
3. 选择 `main` 分支和 `/ (root)` 目录
4. 等待部署完成后访问 `https://你的用户名.github.io/仓库名/`

## 默认全景图

项目包含一张默认的 rogland_clear_night_4k.exr 全景图。
